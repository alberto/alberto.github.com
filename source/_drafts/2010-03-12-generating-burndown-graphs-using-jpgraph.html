Jgraph is a little app I use to generate our burndown graphs. Here is a little how-to use it.
First install the application:
sudo aptitude install jgraph

Create a script to execute it:


#!/bin/bash

jgraph -P -L burndown-1-week.jgr | ps2pdf - burndown.pdf
