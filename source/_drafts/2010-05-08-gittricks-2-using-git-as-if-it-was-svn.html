So you are migrating from svn and want to use git the same way you used to do things in svn? Not ready to mess with the index and local branches?

Use this simple alias:

[alias]
        cosh = !sh -c 'git add -A && git commit -m \"$1\" && git push' -

Now you can just use:

git cosh "Commiting like it is 1999"
